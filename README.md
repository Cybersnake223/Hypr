## ✨ Features

<div align="center">

| | |
|:---:|:---|
| 🔮 | **Material You theming** — Matugen derives a full color palette from your active wallpaper and applies it everywhere |
| 💾 | **Safe installer** — every overwritten file is backed up to a timestamped directory before anything is touched |
| 🎛 | **Modular installs** — pick exactly which modules to deploy: configs, scripts, icons, themes, fonts, or root dotfiles |
| 🪄 | **Dry-run mode** — preview every single action before committing a single byte |
| 🐚 | **Shell-aware PATH patching** — detects Zsh, Bash, Fish, Ksh, and falls back to POSIX `.profile` |
| ↩️ | **One-command undo** — restore your original configs with `./install.sh --uninstall` |
| 🔤 | **Bundled fonts** — JetBrains Mono Nerd, Font Awesome, Icomoon Feather, Nerd Symbols, installed and cached |
| 🌐 | **Zen Browser CSS** — custom `userChrome.css` and `userContent.css` tuned to match the Viper aesthetic |
| 📜 | **50+ custom scripts** — all automatically marked executable and added to `PATH` |

</div>

## 📦 Prerequisites

> [!IMPORTANT]
> The installer checks for these packages and warns about anything missing. It will **not** block installation unless core system utilities like `cp` or `find` are absent.

### Core packages

```bash
yay -S hyprland waybar foot kitty zsh rofi-lbonn-wayland-git mako       \
        hyprlock matugen-bin btop yazi fastfetch neovim starship          \
        cava cmus mpv nautilus zen-browser-bin aria2 advcpmv
```

### Fonts

The `.fonts/` directory is bundled and installed automatically. To install manually:

```bash
yay -S ttf-jetbrains-mono-nerd ttf-font-awesome nerd-fonts-symbols-only
fc-cache -f
```

## ⚡ Installation

> [!CAUTION]
> Always run `--dry-run` first on an existing setup. The installer backs up files it overwrites, but you should confirm what it touches before committing.

### Quick start

```bash
git clone https://github.com/Cybersnake223/Hypr
cd Hypr
chmod +x install.sh
./install.sh --dry-run   # preview first
./install.sh             # run when happy
```

### What happens during install
